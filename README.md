Para iniciar este proyecto deberías tener instalado jekyll
(que generalemente se instala con `gem install jekyll bundler` o con
las [instrucciones oficiales](https://jekyllrb.com/docs/)) y luego
ejecutar el comando:

```
make ejecutar
```
